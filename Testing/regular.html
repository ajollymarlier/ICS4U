Welcome to the Floyd-Warshall Algo Presentation!

The Floyd-Warshall Algorithm is an algorithm designed to discover the shortest paths between all possible pairs of vertices in a graph. Unlike other shortest path algorithms (like Dijkstra and Bellman-Ford), the Floyd-Warshall Algorithm allows for negatively-weighted edges.

let V = number of vertices in graph 
let dist = V x V array minimum distances 
for each vertex v 
  dist[v][v] = 0 
for each edge (u,v) 
  dist[u][v] = weight(u,v) 
for k from 1 to V 
  for i from 1 to V 
    for j from 1 to V 
      if dist[i][j] > dist[i][k] + dist[k][j] 
        dist[i][k] = dist[i][k] + dist[k][j]
